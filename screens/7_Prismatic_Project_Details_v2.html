The Challenge

How do we capture the feeling of a dream that dissolves the moment you wake up? This project was born from a desire to create digital artifacts that are mathematically perfect yet visually unstable.

Traditional 3D rendering focuses on permanence. I wanted to build a system where the "geometry" is merely a suggestion—a probability field shaped by the viewer's interaction.

Process Fragment #04

"The code doesn't define the shape; it defines the rules for the shape to break itself."

Technological Bridge

To achieve the prismatic refraction effect, I developed a custom shader that simulates light dispersion through non-Euclidean surfaces. 

// Dispersion Fragment Shader Fragment
void main() {
  vec2 uv = vUv;
  float dispersion = 0.015;
  vec4 r = texture2D(tDiffuse, uv + vec2(dispersion, 0.0));
  vec4 g = texture2D(tDiffuse, uv);
  vec4 b = texture2D(tDiffuse, uv - vec2(dispersion, 0.0));
  
  gl_FragColor = vec4(r.r, g.g, b.b, 1.0);
}
